# Module 1 — The Robotic Nervous System (ROS 2)

Welcome to Module 1 of the AI-Native Textbook. This module covers the fundamentals of Physical AI and Sensors, ROS 2 basics, and Python Agent Integration.

## Curriculum Overview

This module spans 3 weeks and covers:

- **Week 1**: Introduction to Physical AI and Sensors (LIDAR, IMUs)
- **Week 2**: ROS 2 Fundamentals — Nodes, Topics, Services, Packages
- **Week 3**: Python Agent Integration with ROS Controllers + URDF Modeling

## Prerequisites

- Basic understanding of programming concepts
- Familiarity with command line interface
- Python knowledge recommended

## Learning Objectives

By the end of this module, you will understand:
- The fundamentals of Physical AI and sensor technologies
- ROS 2 architecture and core concepts
- How to integrate Python agents with ROS controllers
- URDF modeling for robotic systems
