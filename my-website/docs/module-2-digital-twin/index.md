# Module 2 — The Digital Twin (Gazebo & Unity)

Welcome to Module 2 of the AI-Native Textbook. This module covers physics simulation in Gazebo and high-fidelity rendering in Unity.

## Curriculum Overview

This module spans 2 weeks and covers:

- **Week 4**: Physics Simulation in Gazebo — Gravity, Collisions
- **Week 5**: High-Fidelity Rendering in Unity + Sensor Simulation

## Prerequisites

- Completion of Module 1
- Understanding of basic physics concepts
- Familiarity with simulation environments

## Learning Objectives

By the end of this module, you will understand:
- Physics simulation principles in Gazebo
- Gravity and collision systems
- High-fidelity rendering techniques in Unity
- Sensor simulation for robotics applications
