# Module 3 — The AI-Robot Brain (NVIDIA Isaac™)

Welcome to Module 3 of the AI-Native Textbook. This module covers NVIDIA Isaac simulation and AI integration for robotics.

## Curriculum Overview

This module spans 3 weeks and covers:

- **Week 6**: NVIDIA Isaac Sim — Photorealistic Simulation
- **Week 7**: Isaac ROS — Hardware-Accelerated VSLAM + Nav2
- **Week 8**: Isaac Sim for Reinforcement Learning — Advanced Tooling

## Prerequisites

- Completion of Modules 1 and 2
- Understanding of AI and machine learning concepts
- Familiarity with NVIDIA development tools

## Learning Objectives

By the end of this module, you will understand:
- NVIDIA Isaac Sim for photorealistic simulation
- Hardware-accelerated VSLAM and Nav2 systems
- Reinforcement learning applications in robotics
- Advanced tooling for AI-robot integration
